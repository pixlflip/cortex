# Hello Cortex

This is a sample note in a sample vault. It exists so you can run Cortex
immediately — `cortex check`, `cortex serve`, and the MCP tools all have
something real to read.

## What is this vault?

A vault is just a folder of Markdown notes. Cortex reads it through the
filesystem (fast, deterministic) and exposes scoped tools over MCP so AI clients
and humans can retrieve from it safely.

## Replace me

Point `vault.path` in `cortex.yaml` at your own notes folder, or drop your files
in here. Then run `cortex init` to create the git audit baseline.
