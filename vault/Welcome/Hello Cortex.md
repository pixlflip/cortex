# Hello Cortex

This is a sample note in a sample Obsidian vault. It exists so you can run Cortex
immediately — `cortex check`, `cortex serve`, and the MCP tools all have
something real to read.

## What is this vault?

The vault is a normal Obsidian vault: Markdown notes, YAML frontmatter, folders,
links, and direct human editability. Cortex reads it through the filesystem
(fast, deterministic) and exposes scoped tools over MCP so AI clients and humans
can retrieve from it safely.

## Replace me

Point `vault.path` in `cortex.yaml` at your own Obsidian vault, or drop your
Obsidian Markdown files in here. Then run `cortex init` to create the git audit
baseline.
